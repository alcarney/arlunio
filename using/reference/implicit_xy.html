Implicit XY¶

Introduction¶

Todo

Move this exposition into the mathematics section when it has been fleshed out.

Rather than draw a single shape the ImplicitXY shape allows you to draw an entire familiy of curves. Those that can be defined using the following relation.

\[f(x, y) = 0\]

where \(x\) and \(y\) are the standard x, y coordinates.

A few examples of such shapes include

A Circle

\[f(x, y) = x^2 + y^2 - r^2\]

A Straight Line

\[f(x, y) = mx + c - y\]

A Parabola

\[f(x, y) = x^2 - y\]

The list goes on. However if we were to use the condition \(f(x,y) = 0\) to define our curves we would never see them as curves have no area. Instead we artificially give our curves some area so that we can draw them with stylo.

We do this by changing the condtion to sat that as long as a point is within a certain distance of the curve we will color it. Mathematically we choose some number \(e > 0\) and pick the points that satisfy the following

\[|f(x, y)| < e\]

Basic Usage¶

To get started with the ImplicitXY shape all we need to do is provide it with an f. Since we already have Circle and Line shapes built into stylo we will use the parabola from above as an example

from stylo.color import FillColor
from stylo.shape import ImplicitXY
from stylo.image import SimpleImage

def f(x, y):
    return x*x - y

color = FillColor()
parabola = ImplicitXY(f)
image = SimpleImage(parabola, color)

Drawing Options¶

You can control how the curve is drawn by using the following arguments

  • pt: This controls the thickness of the curve. (The value of this argument is used as the value of \(e\) we defined in the introduction)
  • above: This shades in the area “above” the curve
  • below This shades in the area “below” the curve

Note

above and below might not behave in the way you would expect depending on the shape of your curve. If you don’t see the result you expected, try using the other argument

from stylo.color import FillColor
from stylo.shape import ImplicitXY
from stylo.image import LayeredImage

def f1(x, y):
    return 2*x*x + 0.6 - y

def f2(x, y):
    return x*x - y

def f3(x, y):
    return x*x/2 - 0.6 - y

color = FillColor()

above = ImplicitXY(f1, above=True)
curve = ImplicitXY(f2, pt=0.2)
below = ImplicitXY(f3, below=True)

image = LayeredImage()
image.add_layer(above, color)
image.add_layer(curve, color)
image.add_layer(below, color)
