Stylo.Testing.Shape¶

class stylo.testing.shape.BaseShapeTest[source]¶

Bases: object

A base class for testing Shape implementations.

When writing your test case for a new Shape implementation you need to declare it as follows.

from unittest import TestCase
from stylo.testing.shape import BaseShapeTest

class TestMyShape(TestCase, BaseShapeTest):
    ...

Note

The order in which you write the classes is very important.

You also need to define the setUp method to set the shape attribute equal to an instance of your shape implementation.

def setUp(self):
    self.shape = MyShape()

Then in addition to any tests you write, your Shape implementation will be automatically tested to see if it passes the checks defined below.

test_add_transform()[source]¶

Ensure that when transforms are added to the list that they are added to the front of the list.

test_apply_transform()[source]¶

Ensure that when a shape carries transforms that they are applied.

test_apply_transform_none()[source]¶

Ensure that when a shape carries no transforms that the domain is returned untouched.

test_draw_method()[source]¶

Ensure that the draw method returns a valid shape function. A shape function must

  • Be callable
  • Only take coordinate arguments defined in RealDomain
  • These arguments must match those defined in the parameters property.
test_has_transforms()[source]¶

Ensure that a shape has a _transforms attribute that is equal to the empty list.

test_parameters_property()[source]¶

Ensure that the parameters property returns the coordinates that the shape is defined with. The parameters must

  • Be iterable
  • Be unique
  • Be declared on the RealDomain interface.
test_render_with_args() → None[source]¶

Ensure that a shape can be called with keyword arguments. Each of the arguments should be numpy arrays of the same shape with names corresponding to each of the coordinate names the shape is defined with. The result must:

  • Be a numpy array
  • With shape (height, width)
  • With elements of type bool.
test_render_with_domain() → None[source]¶

Ensure that a shape can be called with a domain and integers representing the width and height in pixels of an image. The result must:

  • Be numpy array
  • With shape (height, width)
  • With elements of type bool.
test_shape_can_be_anded()[source]¶

Ensure that a shape can be ANDed by checking that an instance of ANDedShape is returned when the & operator is used.

test_shape_can_be_inverted()[source]¶

Ensure that a shape can be inverted by checking that an instance of InvertedShape is returned when the ~ operator is used on the shape.

test_shape_can_be_ored()[source]¶

Ensure that a shape can be ored by checking that an instance of ORedShape is returned when the | operator is used.

test_shape_can_be_xored()[source]¶
