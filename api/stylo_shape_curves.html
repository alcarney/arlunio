Stylo.Shape.Curves¶

class stylo.shape.curves.ImplicitXY(f, pt=0.01, above=False, below=False)[source]¶

Bases: stylo.shape.shape.Shape

Define a curve using an implicit function in the \(x\) and \(y\) coordinates.

A point \((x, y)\) lies on the curve defined by the function \(f(x, y)\) if the following holds.

\[f(x, y) = 0\]

However since paths are one dimensional objects and have no area, if we used this rule as it stands we would never see the curve in question. So instead we introduce some error margin \(e > 0\) and say that a point is on the curve if the following holds.

\[|f(x, y)| \leq e\]
