Stylo.Testing._param_factory¶

This module contains functions that automate the process of writing base test classes for ParameterGroup objects.

Time dependent parameter groups should be able to be called as a function, taking the time t as an argument.

Time dependent parameter groups should be able to take numpy arrays as an argument.
