stylo.domain.rectangular.get_real_domain(width, height, scale=2)[source]¶

Given the width and height for an image return a rectangular domain with appropriate bounds for the image’s aspect ratio.

So that shapes are not distorted when mapped onto non-square images the domain they are mapped onto needs to match the aspect ratio of the image otherwise the shape will be distorted in some way.

This function will construct a RectangularDomain domain with \((0, 0)\) at the centre of the image, the overall size of the domain can be controlled with the scale parameter

Parameters:
  • width (int) – The width of the image in pixels
  • height (int) – The height of the image in pixels
  • scale (float) – This constrols the size of the domain and corresponds to the length of the interval \([ymin, ymax]\)
Returns:

An appropriately sized domain.
Return type:

RectangularDomain
