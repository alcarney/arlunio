Stylo.Math.Interpolation¶

stylo.math.interpolation.lerp(a, b)[source]¶

Return a function that will linearly interpolate between a and b.

The returned function will take the value a when passed the value 0 and will increase/decrease linearly to the value b, taking that value at the point 1.

Parameters:
  • a (float) – The value the function should take at 0
  • b (float) – The value the function should take at 1
