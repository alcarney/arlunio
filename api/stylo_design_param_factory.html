Stylo.Design._param_factory¶

A ParameterGroup is used to help define and manage a related set of parameters. For example a “Position” parameter group could contain values for the x and y position of an object. Another “Stickman” property group could contain all the values required to define a stickman with a particular size and shape.

This module much like the stylo.domain._factory module defines a number of functions for declaring new parameter groups of various types. By writing the base classes using code, we can ensure that all property groups follow the same interface.

Currently the following functions are defined

  • define_parameter_group: This defines a base, static parameter group.
  • define_time_dependent_parameter_group: This defines a time-dependent parameter group, that can take some time t and vary in time. It also provides some utilities for visualising these changes over time.

stylo.design._param_factory.pgroup_getitem()[source]¶

Provide the implementation of the __getitem__ method.

One of the key features of a parameter group is the ability to use the dictionary unpacking syntax (**params) to conveniently pass all the values into a function as a single unit. This syntax is enabled by implementing two methods keys and __getitem__, this function provides the implementation of the latter.
